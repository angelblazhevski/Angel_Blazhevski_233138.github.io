Вештини

HTML и CSS

Во текот на средното образование учевме HTML и CSS, затоа и прилично добро владеам со истите. Во трета година средно учествував во натпревар за изработка на веб страна во која освоив трето место.

PHP

Како што ги изучувавме HTML и CSS во средно, исто така го започнавме и PHP. За разлика од претходните два, овој програмски јазик е back-end, кој што имам помало знаење во споредба со нив, бидејќи и помалку го изучувавме.

Java Script

Како во средно, така и на факултет го проучуваме програмскиот јазикот C++, за кој што можам да кажам дека имам добро познавање. Секогаш решавањето на задачи од овој тип ми представуваат предизвик кој што може да биде исцрпувачки размислувајќи ја целата логика на програмата.
